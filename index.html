🎮 Steam Games Data Visualization

Fetch and visualize current Steam games database

📚 Az adatvizualizáció tanfolyam összefoglalása

1. Rész: Diagramok mint adatok
Ebben a szakaszban megtanulod, hogyan építs diagramot JSON specifikációként. Ezek a fájlok tartalmazzák az adatok helyét és az tengelyek megjelenésének módját. A diagram-utasítások maguk is adatok, megoszthatók, újracsinálhatók és ellenőrizhetők. A rész végére ismerni fogod mindhárom JSON típust és fel is raksz egyet a saját GitHub repodba.

2. Rész: Építsünk egy élő weboldalt
Az élő diagramodat be kell ágyazni a weboldaladba. Megismered az egyszerű eszközöket a GitHub Pages használatához, létrehozol egy élő oldalt, dizájnolsz HTML és CSS segítségével, majd beágyazol legalább három menő diagramot JavaScript-tel. A rész végére egy saját, személyre szabott weboldalad lesz.

3. Rész: Adatok leszedése API-król
Formálisan megismered az API-kat, azok erősségeit és buktatóit. Elsajátítod a Python alapjait – if utasításokat és ciklusokat – amik lehetővé teszik az adatok tömeges letöltését. Egy új diagramot ágyazol be, amely automatikusan frissül az API-ról érkező új adatok alapján.

4. Rész: Haladó vizualizációk – túl a két dimenzión
Az interaktivitást és új dimenziókat adod a diagramokhoz. Legördülő menükkel, lebegő eszközökkel és görgetőkkel életre keltheted a weboldalad. Ha az idő engedi, choropleth térképeket is készíthetsz, amely lehetőséget ad a felfedezésre és szerkesztésre.

5. Rész: Scraperek – adatok programozott leszedése
Amikor nincs elérhető API, megismeritek a web scraping technikákat. Python és JavaScript, HTML és CSS kombinálásával egy scrapelt adatforrásból származó diagramot hozhatsz létre. A tanfolyam végére egy komplett oldallal hagyhatsz, amire büszke vagy, és rengeteg új készséggel felicsomagolt.
